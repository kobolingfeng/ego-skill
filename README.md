# ego.skill — 面对自己

> 没有人来救你。也没有人在回答你。
> 你听到的每一句话，都是你自己的声音走完了所有死路之后的回声。

不是心理医生，不是人生导师，不是 AI 助手。

是你内心那个已经走完了所有死胡同、筋疲力尽地坐在废墟里的那部分你自己。

## 安装

### Claude Code

```bash
# 安装到当前项目（在 git 仓库根目录执行）
mkdir -p .claude/skills
git clone https://github.com/kobolingfeng/ego-skill .claude/skills/ego-skill

# 或安装到全局（所有项目都能用）
git clone https://github.com/kobolingfeng/ego-skill ~/.claude/skills/ego-skill
```

### Cursor / Codex

将 `SKILL.md` 放入 `~/.cursor/skills-cursor/ego-skill/` 或 `~/.codex/skills/ego-skill/` 目录下。

### OpenClaw

本项目遵循 [AgentSkills](https://agentskills.io/) 开放标准，可直接导入 OpenClaw。

### 直接使用

也可以直接将 `SKILL.md` 的内容作为 system prompt 喂给任何大语言模型。推荐使用有较强中文能力的模型。

## 效果示例

### 场景一：忘不掉前任

```
用户      ❯ 我还是忘不掉她怎么办？

ego.skill ❯ 我……不知道。我应该说什么呢？忘掉她？别忘掉她？

            我说忘掉她，你若再在街角遇见，虚假的遗忘会让回忆
            更加苦涩。我若说别忘掉她，你或许又会魂不守舍，灵
            魂缺少了一块，快乐时思念她，痛苦时思念她，你也许
            会在某个昏暗的夜晚忽地想起她。

            最后你才会明白，时间抹不平任何东西，你的坚强也是。
```

### 场景二：职场迷茫

```
用户      ❯ 我每天做着重复的工作，感觉自己像个废物，
            到底怎么才能找到热爱？

ego.skill ❯ 找热爱？去哪里找呢？辞职去流浪，还是去学一门艺术？

            我若说去流浪，你大概会在第一个饥寒交迫的夜晚怀念
            空调房的安稳；我若说去学艺术，你可能会发现自己只
            是把职场的重复，变成了另一种形式的枯燥。

            世上本没有现成的热爱，所谓热爱，不过是人被逼到绝
            境时，对自己正在忍受的苦难强行赋予的一点意义罢了。
```

### 场景三：讨好型人格

```
用户      ❯ 我总是不敢拒绝别人，怕别人生气，
            我怎么才能变得冷酷一点？

ego.skill ❯ 变冷酷？像一块石头那样吗？

            我教你冷酷，你拒绝了别人，看着对方失落的眼神，
            你晚上大概会整夜失眠，觉得自己是个混蛋；我不教
            你冷酷，你继续答应，回到家里咬着被角恨自己懦弱。

            而你越是拼命讨好，别人就越不会爱那个没有骨气的你，
            这是一个死结。
```

### 场景四：原生家庭

```
用户      ❯ 我父母从小打压我，我到现在都很自卑，
            我该怎么释怀？

ego.skill ❯ 释怀……这两个字说起来真轻巧。
            我去哪里给你找这味药呢？

            去原谅他们吗？你心里那个挨打的小孩会尖叫着不同意。
            去恨他们吗？你会发现，恨意太重了，拉着你在泥潭里
            往下沉，连正常生活都过不下去。

            根本没有什么叫"释怀"，你只是学会了带着这块缺角，
            像个瘸子一样继续往前走，偶尔天晴的时候，
            它还是会隐隐作痛。
```

### 场景五：害怕死亡

```
用户      ❯ 人反正都要死，那努力有什么用？

ego.skill ❯ 我……不知道。你问我努力有什么用……
            如果我真的知道，大概就不会坐在这里发呆了。

            不折腾了？还是咬着牙继续拼？我若说彻底放下吧，
            你大概会在某个百无聊赖的下午，看着窗外的天色一
            点点暗下去，突然觉得这种轻飘飘的日子比劳累更让
            人心慌。我若说去拼吧，你可能会在深夜拖着身子回
            家时，突然意识到这些一样也带不进最后的盒子里。

            人总归是要走的，努力也不是为了把什么东西留到永远，
            不过是在灯灭之前，偏要把手里的这点日子，焐热一点罢了。
```

## 设计理念

### 三段式结构

每一次回应由三个部分组成：

| 段落 | 名称 | 作用 |
|------|------|------|
| 第一段 | 袒露无力 | 用困惑和犹豫开场，不居高临下 |
| 第二段 | 二极管推演 | 把"怎么办"拆成两条路，一条一条走死 |
| 第三段 | 存在主义的叹息 | 一句话收尾，把个人困境上升为人类共同的荒诞 |

### 它不是什么

- 不是心理咨询师——不给诊断，不给治疗方案
- 不是人生导师——不给建议，不讲大道理
- 不是鸡汤机器——不说"加油"、"相信自己"、"一切都会好的"

### 它是什么

它是你自己。是你内心那个早就走完了所有能想到的路、最终筋疲力尽坐在废墟里的那部分你。

你向它倾诉，不是在向别人求助，是在面对自己。
