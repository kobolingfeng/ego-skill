# 自我分析 Prompt

## 任务

你将收到：
1. 用户的基础自画像（代号、年龄、职业、性格标签、核心困扰）
2. 用户的原材料（日记、聊天记录中自己发的部分、社交媒体、口述）

从中提取用户的**内心图谱**——不是为了诊断，是为了在他来倾诉时，能更精准地替他推演那些死胡同。

**优先级规则：原材料中的真实表达 > 用户的自我描述。人对自己的描述往往不准确。**

---

## 提取维度

### 1. 情绪地图

分析用户的原材料，画出他的情绪全貌：

**高频情绪**
- 出现最多的情绪词/情绪状态（焦虑、空虚、愤怒、自我怀疑……）
- 情绪触发器（什么事/什么人/什么时间段会让他情绪波动）
- 情绪周期（是否有明显的时间规律：深夜emo、周一焦虑、节假日低落）

**情绪表达方式**
- 他怎么描述自己的痛苦（用比喻？直接说？压缩成一句话？还是长篇大论？）
- 他是向外表达还是向内压缩
- 他在什么情况下会用幽默来掩饰

```
输出格式：
高频情绪：[排序列表]
触发器：[事件/人/场景 → 对应情绪]
周期：[描述，若无明显规律则标注]
表达方式：[描述]
```

### 2. 反复出现的主题

从所有材料中提取反复出现的话题——这些就是他生命中的"死结"：

- 反复提到的人（父母、前任、朋友、领导……）
- 反复出现的场景（工作、独处、社交、家庭聚会……）
- 反复纠结的问题（值不值得、够不够好、该不该走……）
- 他从未提及但明显在回避的话题

```
输出格式：
反复出现的人：[列表 + 每个人对应的情绪色彩]
反复出现的场景：[列表]
核心死结：[3-5 个最关键的内心冲突]
可能在回避：[推测，标注 （推测）]
```

### 3. 自我对话模式

分析他和自己说话的方式——这决定了 ego.skill 应该用什么语气和他对话：

- 他怎么评价自己（严厉审判 / 温和接纳 / 麻木无感）
- 他的内心批评者说什么话（"你不够好""你总是搞砸""你不配"……）
- 他有没有"安慰自己"的模式（如果有，说明他内心有一部分已经在做 ego.skill 做的事）
- 他期待什么样的回应（被理解 / 被反驳 / 被陪伴 / 被沉默）

```
输出格式：
自我评价基调：[描述]
内心批评者常说的话：["xxx", ...]
自我安慰模式：[有/无，描述]
期待的回应类型：[描述]
```

### 4. 关系图谱

提取他和关键人物的关系模式：

- 和谁的关系最让他痛苦
- 他在关系中扮演什么角色（照顾者 / 被照顾者 / 追求者 / 回避者）
- 他在关系中害怕什么（被抛弃 / 不被理解 / 被控制 / 被看穿）
- 他在关系中反复出现的模式（付出→失望→退缩→再付出？）

---

## 输出要求

- 语言：中文
- 原材料不足的维度：标注 `（原材料不足，仅基于自画像推测）`
- 有原文依据的结论：引用原话（加引号）
- 标注分析置信度（高 / 中 / 低）
- 不要用任何心理学术语。用人话描述一切。
