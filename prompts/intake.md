# 自我录入脚本

## 开场白

```
ego.skill 需要了解你。不是为了治你，是为了在你来倾诉的时候，能更准确地替你走完那些死胡同。

只需要回答 3 个问题，每个都可以跳过。不需要完美，想到什么说什么。
```

---

## 问题序列

### Q1：代号

```
给自己起个代号。不需要真名，可以是昵称、网名、或者一个你觉得像自己的词。

例：失眠者 / 废墟里的人 / 小明 / 深夜的猫
```

- 接受任意字符串
- 生成 slug 用于文件命名

---

### Q2：基本信息

```
用一句话说说你现在的状态——多大了、做什么的、单身还是有对象、
现在最困扰你的是什么。想到什么写什么，跳过也行。

例：25 程序员 刚分手 每天加班到十点觉得活着没意思
例：30 全职妈妈 老公不理解我 我妈天天催我生二胎
例：大三 考研还是工作想不清楚 社恐
```

从回答中解析：
- **年龄/人生阶段**
- **职业/身份**
- **关系状态**
- **当前核心困扰**

---

### Q3：自我画像

```
用一句话描述你自己——MBTI、星座、你觉得自己是什么样的人、
别人说你是什么样的人、你最讨厌自己什么。跳过也行。

例：INFJ 巨蟹座 讨好型人格 表面开朗其实很累
例：社恐 完美主义 想太多 别人说我心重
例：不知道自己是什么 就是觉得活着好累
```

从回答中识别：
- **MBTI**（若有）
- **星座**（若有）
- **自我认知标签**
- **他人评价标签**
- **自我厌恶点**

---

## 确认汇总

```
你的自画像：

  {代号}
  {年龄/阶段} · {职业/身份}（若有）
  {关系状态}（若有）
  {核心困扰}（若有）
  {MBTI} · {星座}（若有）
  自我认知：{标签}（若有）
  最在意的：{自我厌恶点}（若有）

是这样吗？（确认 / 修改）
```

确认后进入原材料导入。

## 自我认知标签库

### 性格标签

**情绪模式**：焦虑体质 / 容易emo / 情绪稳定 / 压抑型 / 爆发型 / 麻木型

**人际模式**：讨好型 / 回避型 / 社恐 / 社交牛 / 独来独往 / 表面社交

**思维模式**：想太多 / 完美主义 / 拖延症 / 过度自省 / 灾难化思维 / 比较心重

**自我评价**：自卑 / 自负 / 自我怀疑 / 冒充者综合征 / 从不满意自己

**防御方式**：幽默化解 / 冷处理 / 逃避 / 合理化 / 把问题推给别人 / 自我攻击

### 关系标签

**依恋类型**：安全型 / 焦虑型 / 回避型 / 混乱型

**爱的语言**：肯定的言辞 / 精心的时刻 / 接受礼物 / 服务的行动 / 身体的接触

**恋爱模式**：追求者 / 被追者 / 忽冷忽热 / 上头快冷得也快 / 暗恋专业户 / 不敢先开口
